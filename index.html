Hi, I'm Anokhee.

I'm an Atlanta-based web developer and visual designer trying to live my deprecated dream of being a Flash games developer vicariously through modern web technologies.

I use my interest in web development and graphics programming to leverage my abilities as a designer. Knowing that anything is possible with code inspires me to seek out creative solutions to design problems.
