Beautiful Faces

Computer Graphics, Graphic Design



For the inexperienced, learning how to use a tool like Adobe Illustrator can be an uphill battle.

Beautiful Faces aims to simplify the process of creating vector graphics for designers and non-designers alike. It exploits the programmable nature of SVG by allowing users to easily create and export custom vector faces from the browser.

Fig 01. Four Beautiful Faces after being downloaded as SVGs and edited further in Adobe Illustrator.


Using the Software

Using sliders, users can tweak the proportions of a default face to their liking.


Exporting to SVG

The saved image can be saved as an SVG or PNG. If the image is saved as an .svg and opened in a vector graphics editor like Adobe Illustrator or Inkscape, all the vector paths will be retained and fully editable.
