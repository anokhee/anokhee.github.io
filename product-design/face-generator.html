About this Project

As a child, flash games introduced me to the creative potential of computers. While browser-based gaming has lost its cultural relevance, modern web graphics libraries allow us to create anything we can imagine and make it responsive across various types of devices.

To exercise my creative coding skills, I created a face generator inspired by Flash avatar makers. Using just a few vector points, I was able to generate an infinite variety of unique faces, all drawn using code. Here are some examples:




Final Demos

I used Adobe XD and Photoshop to create a prototype video showcasing the final result, following the process outlined below. In addition, I developed a Live Demo using JavaScript's Canvas API to demonstrate the outcome further. The Live Demo was loosely based on the prototype, but includes a few extra features.




Process


01 - a.

Face Design

Sketching

To create the line art faces using functions from the Canvas API, I began by sketching them out. These sketches were especially useful for plotting complex shapes like Bezier curves. Through some experimentation, I was able to generate shapes that closely resembled cute line art faces.




01 - b.

Face Design

Canvas Implementation

Using the sketches as a reference, I utilized the Bezier curve and ellipse functions provided by Canvas to create a "base face." The red dots in the GIF below indicate the control points for the head shape, which allowed me to plan the UI elements that would eventually be used to manipulate these points.

Styling

Mini Style Guide

The second iteration revealed some misconceptions about what users wanted. In early interviews, users emphasized that they wanted more granular interaction data, down to how many times someone hovered a link. But when presented with mockups showing a greater level of detail, they didn't seem to be as interested in granular interaction data as much as they were interested in how this data varied at different points during the product tour.
