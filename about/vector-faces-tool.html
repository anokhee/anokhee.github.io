Vector Faces Tool

Making it easy to draft SVGs from the comfort of your web browser.

Beautiful Faces is an interactive web tool that allows users to edit a vector-based face and export it to the image software of their choice.

Background

A friendly alternative to the Pen tool.

When I was teaching myself how to use Adobe Illustrator, the Pen tool was the bane of my existence. I found it difficult to draw curves exactly, and creating something like a symmetrical face required a lot of tedious copy-and-pasting and exact positioning.

I wanted to create a web-based tool that allows users to quickly draft a symmetrical face, then export it to SVG. After exporting the image to SVG, they can tweak it further in a software like Inkscape or Illustrator. The program is currently a barebones prototype, but I would love to someday create an advanced, (still free) web-based version that provides users with more freedom and enables them to draft custom graphics beyond faces.

Not only is it kind of entertaining to play with, but users are given a sense of ownership over the graphic which they might not get from a using a template or stock vector, even if it's open source.

Final Thoughts

Inclusive default identicons.

Most default profile images feature decidedly male silhouettes with caucasian-looking hairstyles and features. While this may seem inconsequential, it reinforces the idea of the white male user as the "default person", which can inadvertently alienate minority users.

When a randomize() function is applied to the algorithm generating these faces, the results are more gender-neutral, inclusive, and lighthearted than the typical default profile picture.


Facilitating trustworthy interactions between users.

In community-based sites and applications, users are encouraged to upload a profile picture before contributing to public forums or engaging with other members of the community. Unfortunately, the default profile image can sometimes appear casual/non-participatory at best and creepy at worst.

If a web-based vector editing software is built into an application as a "Customize Your Avatar" feature, it provides an easy solution for users who don't feel comfortable uploading a photo of themselves. Without having to search for an image to use as a profile picture, they can quickly create one that resembles themselves.

SVG is superfast jellyfish.

SVG is highly performant and scalable. This has benefits from a software engineering perspective on the front end and back end.

On the front end, SVG is useful because it is programmable with CSS and JavaScript and maintains perfect image quality at any resolution. On the back end, storing SVG is more efficient because SVG ~70% smaller than an identical PNG on average.
